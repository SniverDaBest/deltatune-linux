## Requirements

- [Quickshell](https://quickshell.org/)
- [Playerctl](https://github.com/altdesktop/playerctl)

## Install & run

```sh
git clone https://github.com/ThatOneCalculator/deltatune-linux
sudo make install
deltatune
```
